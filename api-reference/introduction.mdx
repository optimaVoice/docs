## Webhooks (Available Now)

While the full API is in development, webhooks are available for real-time event notifications:

**Available Events:**
- `call.started` - Call begins
- `call.completed` - Call ends successfully
- `call.failed` - Call encounters error
- `action.triggered` - Integration action performed

**Configure webhooks:** Settings → Integrations → Webhooks
